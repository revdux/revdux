# Revdux

A custom client for [Revolt](https://revolt.chat).

## Quick Start

Open up the web app in (link pending).

Or, you can run it locally on your machine:

```sh
git clone https://github.com/revdux/revdux
cd revdux
yarn
yarn dev
```
